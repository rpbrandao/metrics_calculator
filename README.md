# Cálculo de Métricas de Avaliação de Aprendizado 📊

Este projeto faz parte do desafio de fundamentos de Machine Learning. O objetivo é implementar manualmente as fórmulas de avaliação de modelos de classificação a partir de uma matriz de confusão.

## 🚀 Contexto
A avaliação correta de um modelo vai além da Acurácia. Em cenários de **Cibersegurança** ou no **Agronegócio** (minhas áreas de atuação), entender os Falsos Negativos (Sensibilidade) é crítico para o sucesso do negócio.

## 📈 Métricas Implementadas
- **Acurácia:** Performance global do modelo.
- **Precisão:** Exatidão das predições positivas.
- **Sensibilidade (Recall):** Capacidade de identificar casos positivos reais.
- **Especificidade:** Capacidade de identificar casos negativos reais.
- **F-Score:** Equilíbrio entre Precisão e Sensibilidade.

## 🛠️ Tecnologias
- Python 3.x
- Lógica de matriz de confusão

## 🧑‍💻 Como usar
Basta clonar o repositório e executar o script:
`python metrics_calculator.py`
